Week 2: AWS Cloud Linux Install

Task Overview

Access AWS Academy in Canvas, start the Learner Lab, and then set up an AWS VM Ubuntu Desktop.

Purpose

You need access to this content and software, as we will be using it throughout the course.

Instructions

  1. Ensure you completed the prepare assignment and have access to the learner lab.
  2. Complete the AWS VM Ubuntu Desktop Linux setup instructions.
  3. Finish the AWS VM Ubuntu instructions to complete the following tasks.
    • Log into the AWS Academy course at https://awsacademy.instructure.com/.
    • Start the Learner Lab
    • Provision an Ubuntu System in the instructions.
  4. Submit the URL to your Cloud Ubuntu Desktop by the due date.
    • The format should be https://{your instance IP address}:8443

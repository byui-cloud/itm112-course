Local Linux Install

  • Task: Download the needed software and run Linux on a virtual machine on your computer.
  • Purpose: You need this software and Linux running as we will be using it throughout the course.

Please contact the TA or instructor if you have any questions or encounter any problems during installation. Additionally, you can also Google any errors you get or use AI to help you. Click on the PC or Mac instructions below.

Instructions for PC

If you have a MacBook/Apple computer, skip to the next section of these instructions.

Please complete the following steps before completing anything else in the course.

  1. Download the latest version of the following software to your PC/Mac.
    • VirtualBox Base Package (run it as administrator if you get an error)
    • VirtualBox Extension
    • Ubuntu Desktop - The latest LTS version of Ubuntu Desktop is best. Sometimes newer versions may have issues at first. You may need to edit the settings of the Virtual Machine to use a different display driver. If you have troubles and want and older one that is more likely to work, download the 20.04 version.
  2. Install the following software:
    • VirtualBox Platform Package
    • VirtualBox Extension Pack
  3. These settings:
    1. Memory: 4 GB (4096 MB) (Ubuntu Desktop Requirement)
    2. Disk: 25 GB (25000 MB) (Ubuntu Desktop recommended)
    3. Install Ubuntu Desktop

Want it to run faster? Disable Hyper-V on Windows 10 (not applicable on Mac).

      • If Hyper-V is enabled, disable it so VirtualBox can run. Use the following link to disable Hyper-V: Disable Hyper-V, Microsoft
      • If you don't see in your Windows 10 what the screen captures show in the link above, you do not have Hyper-V enabled.
Instructions for Mac

If you have a computer with an Apple CPU (Specifically the M1, M2, M3, M4, etc. chip), follow the instructions below. If you have an Apple CPU with Intel, you could try the instructions below but you may need modify some settings or use the Virtual Box Install Method at the bottom of the page.

  1. Verify that your MAC has an Apple CPU (Or M1, M2, M3, M4, etc. chip).
    • Follow the instructions in the following document to complete this step: Mac computers with Apple silicon.
  2. Download and install Virtual Box or UTM. Pick ONE.
    • VIRTUAL BOX: Find the Download for your chip type (macOS / Apple Silicon hosts for M Chips): https://www.virtualbox.org/wiki/Downloads
    • UTM: Click on "Download" at the following link to install the software: GetUTM.
  3. Download Ubuntu Desktop for ARM Architecture.
    • Download the 64-bit ARM desktop image at the following link: Ubuntu ARM Desktop Image (GET THE ARM ONE: noble-base-arm64.iso).
  4. Create a VM on UTM and install Ubuntu.
  5. Follow steps on screen for the installation with the following modifications:
    1. Memory: 4 GB (4096 MB) (Ubuntu Desktop Requirement)
    2. Disk: 25 GB (25000 MB) (Ubuntu Desktop recommended)
    3. Install Ubuntu Desktop
  6. Black screen on UTM with Mac/Apple M chips?
    1. Try this: Choose Virtualize (not emulate). Install Ubuntu. Then change the There is an issue with the graphics drivers. Go to your VM Setting -> Display. Then Change Emulated Display Card to virtio-ramfb or virtio-gpu-pci. You might need to click on the icon in the top right corner once you have the VM selected. Go do Display and click the drop down and change from either virtio-ramfb or to this virtio-gpu-pci.

Prepare: AWS Account

Task Overview

Register and access AWS Academy in Canvas and start the Learner Lab.

Purpose

You need access to this content and software, as we will be using it throughout the course.

Instructions

  1. Ensure you have been invited to join the AWS Academy Learner Lab. You should have received an email in your BYUI inbox. If you have not, contact your instructor.
  2. Follow the AWS VM Ubuntu Desktop Linux setup instructions.
  3. Finish the AWS VM Ubuntu instructions to at least start the Learner Lab and complete the following tasks.
    • Log into the AWS Academy course at https://awsacademy.instructure.com/.
    • Start the Learner Lab and take a screenshot
    • Want to keep moving ahead for your next assignment? Provision the Ubuntu System in the instructions.
  4. Submit a screenshot of your Learner Lab dashboard running with the Green AWS button.

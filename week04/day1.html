Part 1: Update your AWS Linux Server

Scenario: You are a Linux system administrator responsible for maintaining a Linux server and ensuring that it is up to date and secure.

# Update your AWS Linux server:

# Make sure you have enough hard drive space

df -h

# Get updated info:

sudo apt-get update

# Get security and app updates:

sudo apt-get upgrade

# Kernel upgrade (minor OS version upgrades):

sudo apt-get dist-upgrade

# Remove old packages

sudo apt autoremove

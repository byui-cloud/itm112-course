Week 8: Mid-term: Capture the Flag (SSH into new server & Find image files)

Overview: In Cybersecurity, Capture the Flag is a common cybersecurity event to find hidden files on a server. In this activity, you will SSH into a Linux server and search for eight image files of country flags. The names of the files follow this format: {Country}-NNNN, where {Country} is a country (e.g., Peru) and NNNN is a four-digit year. Files may have an extension (e.g., .png) or may not. A file name or its directory may be hidden (name begins with a dot).

Connect to the Server

  • On your laptop (not the Linux VM nor AWS Linux), open a shell (Windows: powershell) (Apple/Mac: terminal) and connect with SSH:
    ssh student@52.38.245.170
  • When prompted, use the password: Letmeinplease12345!

Goal

  • Use Linux commands to find the full path to each target file without manually browsing the whole filesystem.
  • Each question of the quiz will give hints listing the World Cup runner-up, third place, and the host country. From provided FIFA websites, determine the winner (champion) and the year from those hints, then search for a file named like WinnerCountry-YYYY.
  • Here are some websites with winners through the years:  FIFA World Cup History,  List of FIFA World Cup Finals,  List of Winners with Location
  • Answer each question with the complete absolute path to the file you found, e.g., /home/user1/peru-1815.jpg (Linux uses forward slashes /).
  • The quiz has multiple attempts for you to try again if needed.

Hints & References

  • You may use websites that list FIFA World Cup winners, finals, and locations to decode the hints.
  • Once you know the country and year, try using locate or find to discover the path.
  • Remember that names are case-sensitive. Country names start with an uppercase letter.

Commands You May Need

  • locate
  • find
  • grep
  • Hidden directories start with a dot.
  • Use AI or Google to help you as needed.

Example Workflow

  1. From the question’s hints (Runner-up, Third place, Host), identify the winner and the year.
  2. Run: find, grep, or locate WinnerCountry-YYYY. If nothing appears, try a different search.
  3. Copy the full path you discover for your answer.

Deliverables for CTF

  • For each of the questions, provide the full absolute path to the discovered file.

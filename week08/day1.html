Useful Commands

  1. Install Docker
    sudo apt install docker.io
  2. Check Docker version
    sudo docker version
  3. List Docker images
    sudo docker image ls
  4. Download the bulletin board project
    
      cd ~
      git clone https://github.com/dockersamples/node-bulletin-board

    View project on GitHub

  5. Change into the app directory
    cd node-bulletin-board/bulletin-board-app
  6. Build the Docker image

    Make sure to include the period (.) at the end of the command.

    sudo docker build --tag bulletinboard:1.0 .
  7. List Docker images again
    sudo docker image ls
  8. Run the Docker container

    This publishes the app on port 8000 (mapped to container port 8080).

    sudo docker run --publish 8000:8080 --detach --name bb bulletinboard:1.0
  9. Check running Docker containers
    sudo docker ps
  10. Allow inbound traffic on AWS

    Go to EC2 → Instances → Select your instance → Security Group → Edit inbound rules.
    Add a Custom TCP rule for port 8000 with source 0.0.0.0/0, then save the rules.

  11. Access the bulletin board website

    Open your web browser and go to http://<your-ec2-public-ip>:8000 to see the bulletin board app.

    Replace <your-ec2-public-ip> with your Public IPv ip address from your AWS instance.

    Make sure to use http and NOT https.

Extra Mile: Docker NGINX and OS Check Instructions

  • See what version of Linux the Docker container is running (by SSH into the Docker): 
    • sudo docker exec -it bb sh
    • cat /etc/os-release
  • Download an NGINX Docker image and deploy it on port 8080: 
    • sudo docker pull nginx:latest
    • sudo docker run --publish 8080:80 --detach --name nx nginx
  • Open port 8080 in AWS Security Group inbound rules:
    • Go to EC2 → Instances → Select your instance → Security Group → Edit inbound rules.
    • Add a Custom TCP rule for port 8080 with source 0.0.0.0/0.
  • SSH into the NGINX Docker container: 
    • sudo docker exec -it nx sh
  • Check the OS of the NGINX Docker container: 
    • cat /etc/os-release

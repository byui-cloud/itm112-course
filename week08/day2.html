Exercise

Part 1: 'Find' vs 'Locate'

Scenario: You are an intern and you need to find the passwd file using the locate function. Next, use the find function.

Ask your favorite AI: "Help me find the passwd file using the locate function. Next, use the find function."

  • Ask AI for help as needed:
  • Find the /etc/passwd file using locate.
  • Install locate if necessary.
  • Create a file named with your name, for example: yourname.txt.
  • See if locate can find your new file. Try using find instead.
  • Run the locate database update: sudo updatedb.
  • Now see if locate can find yourname.txt.
  • See if you can find any other .txt files on your system.
  • Try the find command instead.

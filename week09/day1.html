Overview

Part 1: Connect to the Server

  • On your laptop (not the Linux VM nor AWS Linux), open a shell (Windows: powershell) (Apple/Mac: terminal) and connect with SSH:

    • ONLINE CLASSES:

    ssh student@52.38.245.170
  • When prompted, use the password: Letmeinplease12345!

    • IN-PERSON CLASSES:

    ssh pi4@10.15.0.12
  • When prompted, use the password: raspberry

Part 2: Make a file and folder on the ssh server in the /tmp/ folder

Ask your favorite AI if you need help to complete these tasks:

  • # Change directories to the /tmp/ folder.
    • cd /tmp/
  • #Type 'pwd' to see what folder you are in.
    • pwd
  • # Make a directory with yourname
    • mkdir yourname
  • # Use 'ls' to list the files in the /tmp/ folder to confirm your directory is there.
    • ls
  • # Change directories into the folder with the 'cd' command:
    • cd yourname
  • # Create a new file named: yournamefile.html
    • touch yournamefile.html
  • # Use 'ls' to list the files in the /tmp/ folder to confirm your file is there.
    • ls
  • # Edit the new file called yournamefile.html (put your name) and do it using the text editor VIM:
    • vi yourname.html
  • #Now you are using VIM or VI text editor. To edit the file:
  • Type "i" to go into insert mode, which allows you to add text:
    • Press the 'i' key to enter insert mode. Type "My name is ....." (put your name). To exit "insert" mode in VIM, hit ESC on your keyboard. Type ":wq". That will save (write) and quit.
  • If you have problems and need to save without quitting, type: ESC and then ":q!".

Part 3: Remotely copy your file via SCP

  • # Now open another terminal (Windows: powershell) (Mac/Apple: terminal) window (leave the ssh one you were using) from your laptop (not the ssh server, not from AWS, not from Linux) make a folder using your name on your computer:
    • mkdir sshfolderyourname
  • # Go into the folder
    • cd sshfolderyourname/
  • # Use SCP to remotely copy a file to your computer

    • ONLINE CLASSES:

    scp student@52.38.245.170:/tmp/yourfolder/yourname.html
  • # When prompted, use the password: Letmeinplease12345!

    • IN-PERSON CLASSES:

    ssh pi4@10.15.0.12:/tmp/yourfolder/yourname.html
  • When prompted, use the password: raspberry
  • # Use 'ls' to list the files in your folder to confirm your file is there.
    • ls
  • Take a screenshot of the file succesfully copying to your laptop.
  • In your ssh terminal, disconnect.
  • Disconnect from the SSH server
    • exit

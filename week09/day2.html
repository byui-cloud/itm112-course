Part 2: Deploy C++ API Docker Container

Choose a docker sample you want to try: https://github.com/dockersamples From the GCP terminal, launch a sample docker. Here is an example: https://github.com/dockersamples/c-plus-plus-docker

Example commands if you choose the C++ API docker:

git clone https://github.com/dockersamples/c-plus-plus-docker
cd c-plus-plus-docker/
vi ok_api.cpp

Inside the vi text editor, change 'ok' to 'ok yourname':

  • #Now you are using VIM or VI text editor. To edit the file:
  • Type "i" to go into insert mode, which allows you to add text:
    • Press the 'i' key to enter insert mode.

    Press the arrow keys on your keboard to move to 'ok'

    Type yourname next to 'ok' (put your actual name: ex: Bob Jones)

    To exit "insert" mode in VIM, hit ESC on your keyboard.

    Type ":wq". That will save (write) and quit.

  • If you have problems and need to save without quitting, type: ESC and then ":q!".

Build the C++ docker container/image:

docker compose up --build

When you see this message: "Listening on: http://0.0.0.0:8080/" it is running

Click on the "Web Preview" button (above the terminal) and click "Preview on Port 8080" and now you just accessed the API

It should open a new tab in your browser and show: "message: OK yourname"

Part 3: Extra Mile

Find: the Linux version the docker container is running. To do this, run "docker exec -it c-plus-plus-docker-ok-api-1 bash". Once you get the new shell, type 'cat /etc/os-release'. To get out of the bash shell inside your container, type "exit".

Finishing Up

  • Make sure to click the next arrow on each page of the tutorial to see all four parts.
  • Set up Windows file sharing in your AWS/Cloud Ubuntu instance. You must modify the security group to allow inbound Server Message Block (SMB) traffic to your Ubuntu AWS Linux.
  • Open the Security Group. Ask AI if you don't know what port to open.
  • Connect to the Samba share from your local Windows or Mac/Apple computer. Ask AI if you need help.
  • Use your IPv4 instance's IP address to connect.
  • Once you connect, create a text file on the SMB server named {your firstname_lastname}.txt. For the file's content, enter Hello from {your name} and save the changes to the file and add a sentence of your own. Second, download a BYUI logo image and drag and drop it to the Windows file share location.
  • Submit a screenshot of your file using a cat command from your Cloud Ubuntu instance showing the files in the /home/{username}/sambashare directory you created.

Part 9: Update AWS Server

Ask your favorite AI to: "Update your AWS server."

Next Steps

  • Start your AWS Learned Lab and turn on your instance.
  • Connect to your AWS Ubuntu via the DCviewer, ssh with your pem key, or the AWS "Connect" button
  • Confirm you have enough hard drive space for the updates (1GB or more is usually enough):
    • sudo df -h
  • Get package update info:
    • sudo apt update
  • Install the security updates:
    • sudo apt upgrade
  • Install kernel security updates:
    • sudo apt dist-upgrade
  • Reboot if a new kernel was installed (usually do this late at night):
    • sudo reboot
  • Make a backup snapshot of your server prior to a major upgrade.
  • Complete a major OS version upgrade (this could break your AWS DC Viewer):
    • sudo do-release-upgrade
  • Complete the scenario and ask AI for help as needed.
  • Make sure your name is in the scenario or terminal for identity confirmation.
  • Take a screenshot of your work with the date and time.

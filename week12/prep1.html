Prepare: Linux Fun

Overview

Task: Choose a personal project to build in Linux.

Purpose: Learn about Linux on a project of your choice.

Instructions

  • Choose something interesting to you to build in Linux. Ask AI for ideas if needed.
  • Pick where you want to deploy it: AWS, Google Cloud, Local VM Linux, or on an old computer or raspberry pi.
  • Select what technology and the deployment method: local install, docker container, terraform etc.
  • Other ideas? Install a home assistant, AI server, VPN, Pi-hole, Nextcloud, database server, API server, redis, etc.
  • Unsure how? Ask AI or choose a docker image to deploy on your Linux from https://hub.docker.com/ or a terraform script to deploy something https://registry.terraform.io/
